**PoPoSafari** is an open-source, multiplayer RPG set on the massive **PoPo Island**.
Heavily inspired by the classic **Safari Zone** mechanics from the Pokémon series, this game invites players to explore a vast wilderness, strategize to catch wild Pokémon, and complete their collection in a shared online world.
